# _Setup for NC-News-App_

This app has various articles on a variety of topics which can be filtered by, you can also sort by most comments, most votes and date posted. As a user you will be able to vote on both articles and comments as well as add your own comments and delete them if wanted. If the article Id is know you can search for it or you can scroll through all the articles and click on the title to be taken to that specific article.

## Run project locally

```
Git clone repo down - https://github.com/davidjohn290/NC-news-project.git
```

You will need Node.js v14.6.0 to be able to run the project

```
FIRST RUN 'npm install' in the terminal to install the dependencies then RUN 'npm start' to run the project locally
```
